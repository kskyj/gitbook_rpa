# 💎 수익률 기준 설정

{% hint style="success" %}
수익률 계산방법을변경하는 기능입니다
{% endhint %}

<figure><img src="../.gitbook/assets/image.png" alt=""><figcaption></figcaption></figure>

📋**설정 > 스마트비서 > 수익률 기준**

* 매수금 : HTS에서 제공하는 매수금 기준
* 종목별 원금합 : 매매현황판에 있는 종목별 원금 자동합산
* 입력한 투자금 : 매매현황판 상단에 입력한 투자금
