# 📓 매매상세분석 보고서 확인방법

{% hint style="success" %}
* 매매내역 알림은 주문내역  팝업 / 메일(카카오톡) / 텔레그램 알림 을 제공합니다
* 매매상세 보고서는 카톡(메일), 텔레그램에 제공합니다
{% endhint %}

{% hint style="info" %}

{% endhint %}
